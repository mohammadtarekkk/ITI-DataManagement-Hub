# Listing and Viewing Files

## Listing Files (`ls`)
| Option | Description |
|--------|-------------|
| `ls -a` | All files (including hidden `.` files) |
| `ls -l` | Long listing (permissions, size, date) |
| `ls -ld` | Info about the directory itself |
| `ls -R` | Recursive (all subdirectories) |
| `ls -F` | Append type indicators (`/` folder, `*` executable) |

## Viewing File Content
| Command | Description |
|---------|-------------|
| `cat` | Display entire file at once |
| `more` | Page by page (Space = next, b = back, q = quit) |
| `head -n N` | First N lines |
| `tail -n N` | Last N lines |
