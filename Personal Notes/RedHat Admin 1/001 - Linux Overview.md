# Linux Overview

## Free and Open Source Software (FOSS)
Software that grants users the freedom to view, modify, and distribute source code.
- **GPL (GNU General Public License)**: Most common.
- Others: LGPL, Apache, Mozilla, BSD.

## History
- **1969**: Unix created at Bell Labs.
- **1991**: Linus Torvalds created the **Linux Kernel**.
- **Distributions**: Red Hat (RHEL), Fedora, Debian, Ubuntu, Arch Linux.
- **RHEL**: Used by 90% of Fortune 500 companies — secure, reliable, professional support.

## Linux Components
1. **Kernel**: Core of the OS. Manages hardware, drivers, stays in RAM.
2. **Shell**: Interface between user and Kernel. **Bash** (Bourne Again Shell) is the most common.
3. **Terminal**: Environment where commands are typed and output is displayed.
