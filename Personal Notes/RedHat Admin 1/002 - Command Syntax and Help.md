# Command Syntax and Help

## Command Format
```
command [options] [arguments]
```
- Use `;` to run multiple commands on one line.
- `Ctrl + C`: Interrupt a running command.
- `Ctrl + D`: Terminate input / logout (EOF).

## Getting Help
1. **Man Pages**: `man [command]`
   - Sections: Name, Synopsis, Description, Bugs.
   - `man -k [keyword]`: Search for commands by keyword.
2. **Whatis**: `whatis [command]` — one-line description.
3. **Help**: `[command] --help` — built-in quick reference.

## File Naming Rules
- Max length: **255 characters**.
- **Case sensitive**: `File1 ≠ file1`.
- No mandatory extensions.
