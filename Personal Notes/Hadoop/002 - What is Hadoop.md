# What is Hadoop?

Apache Hadoop is an open-source, Java-based framework that manages **distributed storage** and **distributed processing** for big data applications. It allows clusters of commodity hardware to store and process massive datasets.

## Core Components
Hadoop consists of three main modules:
1. **[[005 - HDFS Architecture|HDFS]]** (Hadoop Distributed File System): The storage layer.
2. **[[024 - YARN Architecture|YARN]]**: The resource management layer (Yet Another Resource Negotiator).
3. **[[016 - MapReduce Concepts|MapReduce]]**: The processing engine.

## History
- **2002**: Doug Cutting started Apache Nutch (search engine crawler).
- **2003-2004**: Google published papers on **GFS** (Google File System) and **MapReduce**.
- **2006**: Doug Cutting joined Yahoo! and split the distributed computing part of Nutch into a new project named **Hadoop** (named after his son's toy elephant).
- **2008**: Hadoop became a top-level Apache project and won the Terabyte Sort Benchmark.
