# What is Data Warehouse?

A **Data Warehouse (DWH)** is a centralized repository that stores data from multiple sources (databases, APIs, files) to support business analysis and decision-making.

## Definition
> "A data warehouse is a subject-oriented, integrated, time-variant, and non-volatile collection of data in support of management's decision-making process." — **Bill Inmon** (Father of DWH)

## Purpose
- **Separate Analytics from Operations**: Moves complex analytical queries away from transactional systems (OLTP) to avoid performance impact.
- **Unified View**: Combines data from isolated silos into a single source of truth.
- **Historical Analysis**: Stores historical data for trend analysis, unlike operational systems that often only keep current states.

## History
- **1970s-80s**: Emergence of decision support systems.
- **1990s**: Inmon and Kimball formalized Data Warehousing concepts.
- **Modern**: Evolution into Cloud DWH, Data Lakes, and Lakehouses.
