# RedHat Linux Labs

## Description
This directory contains the completed lab assignments for the **RedHat System Administration I** course. The assignments are documented within PDF submissions detailing fundamental foundational commands and Linux navigation structure.

## Implemented Workflows
- **Basic Commands**: Explored the differences between reading commands (`cat` vs `more`) and deletion utilities (`rm` vs `rmdir`).
- **File System Handling**: Practiced creating hierarchy directories and actively removing non-empty directories securely via recursive flags (`rm -r`).
- **File Manipulation**: Successfully moved, copied, and renamed critical files (e.g., copying `/etc/passwd` to the home directory as `mypasswd`).
- **Navigation & Inspection**: Listed diverse ways to navigate paths (such as utilizing `$HOME` environment variables natively) and evaluated commands that inspect file bounds directly (`head` / `tail`).
