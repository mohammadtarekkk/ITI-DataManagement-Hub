# Network Labs

## Description
This directory contains the completed networking lab assignments tracking foundational logic on network configurations, port connectivity, and protocols.

## Implemented Workflows
The assignments (contained natively in `.docx` formats) cover detailed steps and verifications for the following concepts:
- **IP Configuration**: Manually configuring IP addresses and managing routing configurations using standard `ipconfig` utilities.
- **Diagnostics**: Validating active nodes utilizing ICMP packets (`ping`).
- **Protocols & Addresses**: Deep diving into MAC addressing layers and managing the ARP (Address Resolution Protocol) tables.
- **Remote Access & File Transfer**: Establishing Remote Desktop Connections securely and utilizing `FTP` over standard command lines and GUI (Filezilla).
- **Security & Ports**: Operating explicit Port Scans and securely overriding network limits physically by altering Windows Defender Firewall rules (e.g., explicitly adding Ping Outbound rules).
