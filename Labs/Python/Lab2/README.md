# Python - Lab 2

## Description
This directory contains the submission for the second Python lab, demonstrating Object-Oriented Programming (OOP) principles in Python.

## Implemented Logic
- **DataJob Class** (`DataJob.py`): Implements a class representing a batch data job. Demonstrates encapsulation by making the `destination` property private. Includes a static method to keep track of the overall initialized job count, and property getter/setter logic for robust modification.
- **ImaginaryNumber Class** (`ImaginaryNumbers.py`): Demonstrates Python operator overloading (dunder methods) by modeling imaginary numbers. It overrides basic arithmetic operations (`__add__`, `__sub__`, `__mul__`) and the equality operator (`__eq__`) to fully support direct mathematical operations on class instances.
