# Python - Lab 1

## Description
This directory contains the submission for the first Python lab. 

## Implemented Logic
- **8 Queens / Chess Game** (`8_Chess_game.py`): A command-line script that simulates placing 8 queens on a 8x8 chessboard without them attacking each other. It includes:
  - Board creation and visualization functions.
  - Robust user input validation logic to map rows and columns correctly.
  - Core game logic to calculate and mark all horizontally, vertically, and diagonally attacked squares when a queen is placed.
