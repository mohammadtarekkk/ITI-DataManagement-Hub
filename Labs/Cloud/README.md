# Cloud Labs

## Description
This directory contains the completed lab assignments for the Cloud computing module. The assignments are documented within PDF submissions containing step-by-step implementations and evidence on AWS (Amazon Web Services).

## Scope & Implementations
- **Lab 1 & 2**: Covered foundational cloud concepts, environment setups, and basic instance initializations.
- **Lab 3**: Implemented actionable Cloud Monitoring workflows, including:
  - Creating and launching an EC2 Instance.
  - Creating specific Alarm Topics via Simple Notification Service (SNS) and establishing active Subscriptions.
  - Configuring customized Alarm Metrics dynamically (e.g., establishing a trigger when CPU utilization exceeds 70% continuously for 10 seconds).
