# Java II - Lab 4

## Description
This directory contains the fourth lab assignment for the Java II course, covering Database connectivity (JDBC) and persistent data tracking.

## Implemented Logic
- **personDB**: A local embedded database schema (often Derby/H2) structuring table storage specifically for querying personnel data.
- **lab4**: The Maven-backed Java project implementing the Database Connection string mappings natively retrieving, inserting, and handling persistence data from the configured local DB safely using SQL encapsulation constraints.
