# Java II - Lab 1

## Description
This directory contains the first lab assignment for the Java II course. The assignment covers foundational GUI construction natively using the modern `JavaFX` ecosystem within a Maven structure.

## Implemented Logic
- **JavaFX Architecture**: Configured a Maven project parsing FXML views mapped to `PrimaryController.java` and `SecondaryController.java`. It tests scene switching and basic JavaFX event handling.
