# Java II - Lab 3

## Description
This directory contains the third lab assignment for the Java II course, demonstrating practical client-server network architecture and socket programming using Java.

## Implemented Logic
- **ChatServer**: The robust server-side implementation listening to an established port and handling concurrent incoming socket connection streams natively.
- **chatclient_gui**: A Maven-backed `JavaFX` / Swing client application attaching to the hosted socket architecture to simulate real-time chat operations securely.
