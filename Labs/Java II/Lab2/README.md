# Java II - Lab 2

## Description
This directory contains the second lab assignment for the Java II course, involving a more complex, practical JavaFX build.

## Implemented Logic
- **JavaFX Text Editor**: The `TextEditorController.java` acts as the backing component for a fully functional GUI text editor application featuring real-time typing events, menu bindings, or saving semantics natively routed via JavaFX controllers and FXML views.
