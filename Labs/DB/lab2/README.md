# DB - Lab 2

## Description
This directory contains the submission for the second Database (DB) lab, focusing on basic DML operations (Data Manipulation Language) and fundamental selections over a company database.

## Questions Addressed & Implemented Logic
- **DML Operations**: 
  - Inserting new employee records.
  - Adding a new department and calculating/assigning its manager.
  - Updating department managers and employee supervisors.
  - Safely deleting an employee by cascading updates to dependent tables (dependents, department, works_on).
  - Updating employee salaries (e.g., adding a 20% bonus).
- **Basic Queries**: 
  - Retrieving full employee data and calculating annual commissions automatically.
  - Filtering by salary thresholds (>1000 monthly, >10000 annually) and gender.
  - Finding specific departments based on manager IDs.
  - Extracting projects controlled by specific departments.
