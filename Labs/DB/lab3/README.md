# DB - Lab 3

## Description
This directory contains the submission for the third Database (DB) lab, which covers complex `JOIN` operations, Unions, aggregations, and subqueries.

## Questions Addressed & Implemented Logic
- **Joins**: Extracting combined data from multiple tables (e.g., Department, Project, Employee, Dependent, Works_on).
- **Unions**: Combining full names and genders of both employees and dependents into a single result set.
- **Pattern Matching**: Filtering projects starting with specific letters (`like 'A%'`) or located in specific cities using the `IN` operator.
- **Aggregations & Grouping**: Extracting max, min, and average salaries per department. Summing total weekly hours spent per project.
- **Complex Subqueries**: Finding the department with the smallest overall employee ID, and selecting departments where the average salary is below the company's overall average salary.
