# Intro to Programming - Lab 2

## Description
This directory contains the second lab for the Intro to Programming course, expanding on basic syntax to cover structured data and OOP mechanics.

## Implemented Logic
- **Array Manipulations (`Arrays` package)**: Solves several common array algorithms, including:
  - Finding the average of elements in an array greater than a specific value.
  - Deep-copying an array.
  - Finding the second-largest element in an array.
  - Processing and counting positive/negative numbers.
  - Replacing negative values in an array with zeros.
  - Summing elements from two different arrays at the exact same index location.
- **Java Methods (`Methods` package)**: Contains several static method implementations (`M01` through `M06`) focusing on modular code execution.
- **Financial Modeling (`Loan.java`)**: A practical implementation modeling a banking loan logic, calculating monthly and total payments based on the principal amount, interest rate, and terms in years.
