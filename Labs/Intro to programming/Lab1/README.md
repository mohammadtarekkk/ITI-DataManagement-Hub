# Intro to Programming - Lab 1

## Description
This directory contains the first lab for the Intro to Programming course, featuring introductory Java programs that solidify fundamental logic handling.

## Implemented Logic
The source code covers basic control flow mechanisms:
- **Conditionals**: Basic conditional logic (e.g., checking if an integer is divisible by 6 in `Condition1.java`).
- **Loops**: Basic loop iterations handling arrays of inputs, such as taking 5 numbers from the user and counting/summing the positive, negative, and zero inputs dynamically (`Loops1.java`).
- **Problem Solving**: Various mathematical logic problems translated into Java syntax (e.g., evaluating complex algebraic/geometric mathematical expressions using `Math.sqrt` in `Problem1.java`).
