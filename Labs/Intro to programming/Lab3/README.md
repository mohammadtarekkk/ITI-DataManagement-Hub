# Intro to Programming - Lab 3

## Description
This directory contains the third lab for the Intro to Programming course. This lab dives deeply into Object-Oriented Programming (OOP) concepts by architecting full structural models for real-world scenarios.

## Implemented Logic
The source code (`Lab3/src` directory) contains packages separating logic for distinct business models:
- **HR**: Simulates Human Resources structures (e.g., employee management, tracking).
- **HotelTour**: Simulates a booking or reservation-like model.
- **PatientsDoctorModel**: Implements a classic clinical system architecture matching doctors to their patients.
- **VocationPackage**: Models holiday or vacation packages.
