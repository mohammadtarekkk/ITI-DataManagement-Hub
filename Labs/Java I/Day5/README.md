# Java I - Day 5

## Description
This directory contains the fifth lab assignment for the Java I course, focusing strongly on Threading models and frame-based animations within Java Applets/GUIs.

## Implemented Logic
- **AnimationBallGUI**: Demonstrates multithreading combined with graphics by rendering a ball bouncing across the canvas boundaries seamlessly.
- **DateGUI**: Uses threading or timers to continually fetch and visually render the current system date/time in the user interface.
- **TextBannerGUI**: Replicates a scrolling "marquee" or banner tracking logic using threads to move a string coordinate iteratively across the GUI screen.
