# Java I - Day 3

## Description
This directory contains the third lab assignment for the Java I course, transitioning into string tokenization and the introductory building blocks of Java windowing and Graphical User Interfaces (GUI).

## Implemented Logic
- **Calculate**: Logic parsing mathematical inputs to perform calculations, possibly interacting with command line arguments.
- **GUI**: A foundational Java Applet or AWT/Swing interface establishing a simple window structure.
- **StringSplitter**: Logic focused on string manipulation, tokenization, and splitting large inputs based on specific delimiters.
