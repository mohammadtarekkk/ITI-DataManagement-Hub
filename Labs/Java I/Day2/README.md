# Java I - Day 2

## Description
This directory contains the second lab assignment for the Java I course. The programs demonstrate class interactions and foundational OOP principles like inheritance.

## Implemented Logic
- **Parent**: Demonstrates core inheritance implementation between a parent and child class structure.
- **Shape**: A foundational structure modeling geometric shapes, likely utilizing abstract methods or polymorphism to calculate area/perimeter.
- **Stars**: Implementing loops and logic inside a class to print structured star representations or patterns to the console.
