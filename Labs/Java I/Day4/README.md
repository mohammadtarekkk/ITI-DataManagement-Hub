# Java I - Day 4

## Description
This directory contains the fourth lab assignment for the Java I course, advancing into GUI rendering contexts and styling.

## Implemented Logic
- **FontGUI**: Shows how to manipulate GUI font rendering, styles, sizes, and specific font families natively in Java layout managers.
- **LampGUI**: A GUI application illustrating visual states, changing a graphical representation (turning a lamp ON or OFF) directly on a canvas based on user interaction or logic.
