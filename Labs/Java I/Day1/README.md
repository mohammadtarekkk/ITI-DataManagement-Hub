# Java I - Day 1

## Description
This directory contains the first lab assignment for the Java I course. It features introductory Java programs demonstrating fundamental object-oriented encapsulation and logic operations.

## Implemented Logic
- **ComplexNum**: A Java class representing a complex number (with real and imaginary private attributes). It provides static methods to perfectly add (`addNum`) and subtract (`subtractNum`) two complex numbers and prints the corresponding `real+imagi` result.
- **Student**: A classic Java POJO class modeling a Student entity with properties like `name`, `age`, and `track`. It encapsulates these fields using getter and setter methods and includes a `printInfo` routine.
